Personal project aiming to create a chess bot that you can play against.

Backend written in Python, utilizing the Python-Chess library for the chess
mechanics. The backend includes a basic CLI (with limited testing/error-handling)
in order to manually see how the program is working, and play around with the 
functionallity. The frontend is written with React, and the goal is to have this
be the only part the user would actually interact with.
